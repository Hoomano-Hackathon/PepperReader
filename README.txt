
Projet réalisé par "Les Sciences Co + 1" pour le hackathon organisé par Hoomano 

Ce projet repose sur une lecture vocale d'un support écrit (papier), par le robot Pepper.
Il utilise sa caméra supérieure pour prendre une photo du support, sur laquelle il effectue un traitement de l'image, afin d'appliquer un algorithme OCR, tesseract actuellement
